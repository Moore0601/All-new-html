Introduction to Git and Github

Git

Git is the most popular version control system, is a distributed version system meaning each developer has a full copy of the entire project history on their local machine. Also Git is a software that is used to track the different changes made to our code base over time. Changes can be tracked offline, and update are shared by pushing or pulling from a remote repository such as Github

Github

Github is a cloud based platform designed to help developers manage, host and collaborate on project using the Git version controil system. Github provides a web interface for managing Git repository and offers various tools that facilitates team collaboration, code sharing and project management. Github is widely used for both open source and pricate projects ande it integrate samelessly with Git.
